# Arch Setup

> My streamline arch install script for installing KDE or DWM

![Coverpic](./pacman.png)

## Features

-   linux-zen kernel
-   btrfs
-   zram
-   apparmor

## KDE Build

> Minimal KDE with less bloat.

## DWM Build

> Minimal DWM build with less patches.

**DWM PATCHES**

-   3 column Layout
-   Fibonacci Layout
-   Cycle across layouts
-   Move Stack
-   vanity gaps
-   always center

Emoji support by installing libxft-bgra

**DMENU PATCHES**

No patches (Vanilla)

Emoji support by installing libxft-bgra

**SLOCK PATCHES**

-   slock-message

**DWM BAR**

-   Cpu Ram Disk usage
-   Sound
-   Brightness
-   Network
-   Date and Time
